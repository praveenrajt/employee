**Employee**

This project is basic level of ruby on rails crud operation with authentication.

Ruby version **ruby 2.6.3**

Bunlder version  **bundler version 2.1.2**

**Configuration**

`git clone https://github.com/praveenrajt/employee.git`
`cd employee`
`bundle`

**Database creation**
`rails db:migrate`

**Run Program**
` rails s`
